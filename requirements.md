## Project Overview
ReadyRight is an AI-powered readiness assistant that helps citizens evaluate eligibility and preparedness for welfare schemes before submitting applications.

The goal is to prevent avoidable rejection and improve access to public resources.

---

## Functional Requirements

- User can select a welfare scheme category
- System asks adaptive eligibility questions
- AI evaluates readiness based on structured rules
- System predicts eligibility risk
- System explains missing requirements in simple language
- System provides next-step checklist
- Multi-language support (design-level)

---

## Non-Functional Requirements

- Low bandwidth accessibility
- Simple interface for low-literacy users
- No long-term storage of sensitive personal data
- Explainable AI outputs
- Scalable serverless architecture

---

## User Requirements

- Clear eligibility guidance
- Minimal questions
- Easy-to-understand explanations
- Confidence before submission

---

## Constraints

- Prototype scope limited to selected welfare schemes
- Uses publicly available rule data
- Human validation required for rule updates
